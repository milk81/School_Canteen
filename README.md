__Цель проекта__: 

Разработка веб-приложения для автоматизации работы школьной столовой, обеспечивающего удобный заказ и оплату питания, контроль учета продуктов и прозрачную систему отчетности.
Система направлена на повышение эффективности работы столовой, улучшение качества обслуживания учащихся и обеспечение безопасности данных и платежей.

__Целевая аудитория__:
```
  •  Ученики 1–11 классов (основные пользователи)
  
  •  Повара (сотрудники столовой)

  •  Администратор (модерация)
```
  
__Основные функции__:

  __Для ученика__:

    •  Регистрация и авторизация;

    •  Просмотр меню;

    •  Оплата питания (разово или абонемент);

    •  Отметка получения питания;

    •  Указание аллергий;

    •  Отзывы о блюдах.
    
  __Для повара__:

    •  Учет выданных блюд;

    •  Контроль остатков;

    •  Создание заявок на закупку.
    
  __Для администратора__:

    •  Просмотр статистики оплат и посещаемости;

    •  Согласование заявок;

    •  Формирование отчетов;

    •  Контроль расходов.
    
__Технологии__:

    •  Backend: Python + Flask (микрофреймворк для гибкости).
  
    •  Frontend: HTML/CSS, JavaScript, Bootstrap (адаптивный дизайн).
  
    •  Хранение данных: JSON-файлы (users.json, materials.json, forum_topics.json).
  
  
__Уникальность__:

    •  Использование JSON вместо классической БД для упрощения разработки и поддержки.
    
    •  Разделение ролей с четкой системой уровней доступа.
  
    •  Учет пищевых аллергий и индивидуальных предпочтений учащихся.
  
    •  Модерация и фильтрация контента.
